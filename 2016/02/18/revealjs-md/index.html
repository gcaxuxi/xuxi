使用Markdown来制作幻灯片

  • 技术

近来有空，体验了一下如何使用markdown来制作slides。在此之前，我已经用过的制作slides的工具如下：

  • PowerPoint
  • Keynote
  • Latex + Beamer
  • ConText

一般的幻灯片当然PowerPoint和Keynote又方便又好用，可是当制作学术型的幻灯片的时候，要显示大量数学公式，这个时候使用它们就非常头大。用公式编辑器敲公式的感觉真是生不如死，前几日刚写过一个专利书，对此深有体会。

对数学公式友好的自然是LaTeX系列的工具了，但是Beamer实在太丑了，感觉就是上个世纪的东西，一开始的时候用它做slides倒是蛮有成就感的，可是随着时间推移，都不好意思拿这种反扁平风格的东西出来见人了。ConTexT又非常难用，好在物色到了reveal.js这么一个好东西。

reveal.js是一个生成html幻灯片的好东西，虽然不懂html等等语言，好在可以使用markdown，然后利用转换神奇pandoc来生成html文件。

尽管目前用markdown做幻灯片的工具有好多，
但是使用Reveal.js有很多好处：

  • 可选的现代感主题
  • 可选的切换效果
  • 最重要的是方便

配置reveal.js很简单，

  1. 首先需要下载reveal.js
  2. 然后解压把文件夹重命名为revealjs,
  3. 之后在与该目录同级的目录下面创建你自己的markdown文件slides.md，
  4. 之后尽情地用markdown书写，完了以后，可以使用pandoc生成slides.html.
pandoc slides.md -o slides.html -t revealjs -s -V theme=moon -V transition=slide  --slide-level=2
