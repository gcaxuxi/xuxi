Convex Functions

old

假设函数$f$定义在$S$上，

epigraph

$$
\mathrm{epi}\; f=[(x,\mu)|x\in S,\mu\in R,\mu \ge f(x)]
$$

convex function

$f$ is convex iff. $\mathrm{epi}\; f$ is convex as a subset of $R^{n+1}$

concave function

$f$ is concave iff. $-f$ is convex

affine function

$f$ 有限，同时凸和凹，则称为仿射函数

effective domain

凸函数的等效定义域，

$$
\mathrm{dom}\; f=[x|\exists \mu,(x,\mu) \in \mathrm{epi}\;f] = [x|f(x) < + \infty]
$$

$\mathrm{dom}\;f$的维度称为$f$的维度

定理

  • 令$f$为定义在$C$上的函数，$C$是凸集，则$f$在$C$上是凸的，当且仅当，

$$
\forall xy\in C,\;f((1-\lambda)x+\lambda y)\le (1-\lambda)f(x) + \lambda f(y),\;0 < \lambda < 1
$$

这是显然的，因为$f$是凸的当且仅当，

$$
\forall (x,\mu),(y,\nu)\in \mathrm{epi}\; f
$$

有

$$
(1-\lambda)(x,\mu) + \lambda (y,\nu) = ((1-\lambda)x+\lambda y,(1-\lambda)\mu + \lambda \nu) \in \mathrm{epi}\;f
$$

即
$$
f((1-\lambda)x + \lambda y) \le (1-\lambda) \mu + \lambda \nu
$$

  • 琴生不等式：（Jensen）

$$
f(\lambda_1 x_1 + \cdots + \lambda_m x_m)\le \lambda_1 f(x_1) + \cdots + \lambda_m f(x_m)
$$

$$
\lambda_1 \ge 0,\cdots,\lambda_m \ge 0,\;\lambda_1 + \cdots + \lambda_m = 1
$$

  • 若$f$为在开区间$(\alpha,\beta)$的二次可导实值函数，则$f$为凸的当且仅当其二阶导数在$(\alpha,\beta)$上非负。

  • 高维情形下，$f$凸当且仅当其Hessian半正定。

indicator function

$$
\delta(x|C)=
\begin{cases}
0&if\; x\in C\
+\infty &if\; x\notin C
\end{cases}
$$

support function

$$
\delta^*(x|C) = \sup [\langle x,y
\rangle|y \in C]
$$

gauge function

$$
\gamma(x|C) = \inf[\lambda \ge 0| x\in \lambda C]
$$

distance function

$$
d(x,C) = \inf[|x-y||y\in C]
$$

以上这些函数都是凸的，在$C$为凸的时候。

level set

$$
\forall f, \alpha \in [-\infty,+\infty]
$$

若$f$是凸的，则水平集

$$
[x|f(x)<\alpha]\;[x|f(x)\le \alpha]
$$

也都是凸的

positive homogeneous

一个函数称为正齐次，若

$$
f(\lambda x) = \lambda f(x),\; 0 < \lambda < \infty
$$

一个正齐次函数是凸的，当且仅当

$$
\forall xy\in R^n\; f(x+y) \le f(x) + f(y)
$$
